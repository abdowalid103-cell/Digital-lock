# Digital-lock
The Digital Lock using ATmega32 is a security system based on entering a password through a keypad. The ATmega32 microcontroller verifies the code, and if it is correct, the lock is opened using a motor or servo while the status is displayed on an LCD, providing a secure and easy access control system
